# Dashboard de Vendas – Excel

Projeto desenvolvido como parte de um desafio da DIO com foco na criação de um dashboard de vendas utilizando o Excel.

## Dados
Os dados foram fornecidos em um arquivo Excel contendo informações de assinaturas, planos, valores e tipos de renovação.

## Dashboard
O dashboard apresenta indicadores como:
- Receita total
- Quantidade de assinantes
- Receita por plano
- Receita por tipo de assinatura
- Receita por renovação automática

## Ferramenta
- Microsoft Excel
